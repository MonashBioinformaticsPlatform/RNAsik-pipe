Download & Install

Please take a look at the download page.

Prerequisites

  • BigDataScript
  • STAR aligner
  • featureCounts
  • samtools
  • Picard tools
  • gffutils python package, can use (sudo) pip install gffutils
  • RNA-SeQC

System requirements

  • At least 100 Gb of disk space. STAR aligner you will need 100 Gb of disk space during making an index step. The index itself isnâ€™t 100 Gb, but that much space is required.
  • At least 32 Gb of RAM for read alignments

Installation

The easy way: use ansible RNAsik-playbook

The other way:

  • Install BigDataScript

  • Install latest stable RNAsik-pipe release

    1. Locate your *tar.gz file
    2. tar zxvf *tar.gz file
    3. You RNAsik-pipe executable file is located in src directory

  • Install all dependencies

To get developing version: git clone https://github.com/MonashBioinformaticsPlatform/RNAsik-pipe.git
