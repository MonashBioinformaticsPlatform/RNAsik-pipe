RNAsik - The right place to start

Introduction

Why choose RNAsik? Cause it's sik easy. Get list of differentially expressed genes in a single command line. Simply give your FASTQ files to RNAsik-pipe with your reference files and press go ! [Enter] Go now ! RNAsik has been written in BigDataScrip (BDS) language. There are many advantages in using BDS, but main ones are:

  1. One script works on laptop, server and cluster
  2. Checkpoints - restart from where you've left off/li>
  3. Bucket your data - push and pull over http and S3

Quick start

If you have a module system on you server/cluster make sure to module load all required tools e.g module load STAR Need to have all of these tools in your enviroment. Alternatively use ansible RNAsik-playbook to install everything for you and module load RNAsik which will automatically load all the dependencies.

Use RNAsik executable file to run it e.g ./RNAsik. Everything else will depend on what you want to do, some examples below

RNAsik-pipe commands examples

  • To align FASTQ files in raw-data/data directory use
                                                 RNAsik -align star -samplesSheet path/to/samplesSheet.txt -fqDir raw-data/data -fastaRef path/to/yourFASTAfile -threads 15
  • To just get counts on already existing bam files you must have directory bamFiles with all BAM files with either _Aligned.out.bam or _Aligned.sortedByCoord.out.bam postfix as per STAR aligner output.
                                                RNAsik -count -gtfFile path/to/your/annotation/file -threads 15
  • Full RNAsik run
                                                RNAsik -align star -samplesSheet path/to/samplesSheet.txt -fqDir raw-data/data -count -gtfFile path/to/your/annotation/file -prePro -fastaRef path/to/yourFASTAreference-file -threads 15
